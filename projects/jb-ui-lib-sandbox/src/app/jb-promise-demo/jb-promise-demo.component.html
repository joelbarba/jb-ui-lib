This reproduces the native ES6 Promise logic, extending its capabilities:

  • Cancel option --> It is possible to cancel a pending promise
  • Exposed status --> The status is readable at any time, to be checked out of the promise (IJbDeferStatus)
  • Complete --> It exposes a complete (like finally) subscription, available on the then() function too.
  • Resolve / Reject --> It exposes the resolve() / reject() methods externally, so they can be called out of the constructor

It does not use a native promise internally, but reproduces the stack callback chain so it has full controll on the ongoing subscriptions. It can also be chained.
